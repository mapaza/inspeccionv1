Bienvenido (a) al Laboratorio de Inspección Técnica

Utilice los controles del mouse para interactuar con el escenario

  1. Para rotar y desplazarse por las vistas del laboratorio utilice el botón izquierdo del mouse
  2. Para desplazarse hacia la derecha o arriba presione el botón derecho del mouse o las teclas físicas ↑ ↓ → ←
  3. Para acercar o alejar las vistas del laboratorio utilice la rueda del mouse
  • Para iniciar el laboratorio, dirígase al Menú de Actividades ubicado en la parte lateral izquierda y empiece presionando sobre la primera actividad Inspección a Cigueñal
  • Después presione sobre los elementos que se resaltan en color azul para continuar con el proceso
  • Realice el mismo procedimiento para continuar con las siguientes actividades

Objetivos del Laboratorio

  1. Realizar correctamente la inspección visual a diferentes componentes mecánicos para identificar diferentes tipos de desgaste como abrasión, adhesión, erosión, fatiga y corrosión.

  2. Adquirir imágenes especificas de las zonas de desgaste de cada componente para que el estudiante pueda incluir esas imágenes en un informe técnico que permita evidenciar los diferentes tipos de desgaste identificados.

Identificación de Tipos de Desgaste
  • ACTIVIDADES
  • 1. Inspección Visual de Cigueñal
  • 2. Inspección Visual de Pistón
  • 3. Inspección Visual de Bomba Refrigerante
  • 4. Inspección Visual de Pin de Transmisión
  • 5. Inspección Visual de Camisa de Motor

Gire el componente en todas las direcciones para realizar la inspección general y ubicar las zonas con el desgaste característico. Al terminar presione el botón de siguiente para continuar con el laboratorio.
